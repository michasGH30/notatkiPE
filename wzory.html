Natężenie pola elektrycznego

siła elektryczna: \[F=k\frac{q_1q_2}{r^2}\] \(q_1, \ q_2\) - ładunki określone w kulombach [C]
\(r\) - promień w metrach
\(k\) - współczynnik proporcjonalności \(\frac{1}{4\pi\varepsilon_0}\)
\(\varepsilon_0 \approx 8,854 * 10^{-12}\) - przenikalność elektryczna
natężenie pola: \[E=\frac{F}{q_0}\] \(q_0\) - ładunek próbny
po połączeniu: \[E=k\frac{q}{r^2}\] inaczej szybkość zmian ładunku: \[i(t) = \frac{\text{d}q(t)}{\text{d}t}\]

Potencjał i napięcie elektryczne

Napięcie to praca potrzebna aby przesunąć ładunek próbny \(q_0\): \[U=\frac{W}{q_0}\] Praca jest równa sile pomnozonej przez przemieszczenie: \[U=k\frac{q}{r}\]\(V\) - jednostka

Prąd stały

prąd stały - \(I\)

Prąd zmienny

prąd zmienny - \(i(t)\)
\(T\) - okres przebiegu
\(f=frac{1}{T}\) - częstotliwość w hercach [Hz]
napięcie: \(u(t)=\sin t = U_m\sin(\omega t + \phi)\)
\(U_m\) - amplituda, wartość maksymalna/szczytowa
\(\omega\) - prędkość kątowa, pulsacja
\(\phi\) - przesunięcie fazowe, kąt fazowy
natężenie: \(i(t)=I_m\sin(\omega t + \theta_i)\)

Prawo Ohma

\[I=\frac{U}{R}\] \[U=RI\]

Moc elektryczna

jednostka - \(W\) - wat \[p(t)=\frac{\text{d}W(t)}{\text{d}t}\] \[p(t)=u(t)i(t)\]

Napięciowe prawo Kirchhoffa

\[\sum_{k=1}^l \pm U_k = 0\] w kazdym oczku obwodu algebraicznych suma \(l\) napięć jest równa 0

Dzielnik napięcia

\[U_1=\frac{R_1}{R_1+R_2}U\] \[U_2=\frac{R_2}{R_1+R_2}U\]

Prądowe prawo Kirchhoffa

\[\sum_{k=1}^B \pm I_k = 0\] suma wszystkich \(B\) prądów gałęziowych jest równa 0

Dzielnik pradu

\[I_1=\frac{R_2}{R_1+R_2}U\] \[I_2=\frac{R_1}{R_1+R_2}U\]

Rezystor

jednostka - \(\Omega\) - om

odwrotność: konduktancja \(\frac{1}{R}\) - simens

rezystancja: \(R=\rho\frac{l}{A}\)
\(\rho\) - rezystywność materiału
\(l\) - długość przewodu
\(A\) - pole przekroju poprzecznego

Szeregowe połączenie rezystorów

\[R=R_1+R_2+R_3\]

Równoległe połączenie rezystorów

\[\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3}\]

Prąd zmienny

napięcie: \(u(t)=U_m\sin(\omega t + \omega)\)
natężenie: \(i(t)=\frac{u(t)}{R}=\frac{U_m}{R}\sin(\omega t + \theta) = I_m\sin(\omega t + \theta)\)

Postać zespolona

\[U_{R_{rms}} = \frac{U_m}{\sqrt{2}} = RI_{R{rms}}\]

Kondensator

jednostka - \(F\) - farad

pojemność: \(C=\frac{q}{U}\)

odwrotność: elastancja \(\frac{1}{C}\)

Szeregowe połączenie kondensatorów

\[\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2} + \frac{1}{C_3}\]

Równoległe połączenie kondensatorów

\[C = C_1 + C_2 + C_3\]

Prąd zmienny

natężenie: \(i_C(t)=\omega CU_m\sin(\omega t + \theta_u + \frac{\pi}{2})\)
\(I_m = \omega CU_m\)
lub: \(U_m=\frac{1}{\omega C}I_m\)
i \(\theta_i=\theta_u + \frac{\pi}{2}\)

Reaktancja pojemnościowa/opór bierny

\(X_C=\frac{1}{\omega C}\)

Odwrotność reaktancji/susceptancja pojemnościowa

\(B_C=\frac{1}{X_C} = \omega C\)

Postać zespolona

\[-jU_{C_{rms}} = -j\frac{I_{rms}}{\omega C}\]

Cewka

jednostka - \(H\) - henr

\(L\) - indukcyjność

Szeregowe połączenie cewek

\[L=L_1+L_2+L_3\]

Równoległe połączenie cewek

\[\frac{1}{L} = \frac{1}{L_1} + \frac{1}{L_2} + \frac{1}{L_3}\]

Prąd zmienny

\[u_L(t)=L\frac{\text{d}i_L(t)}{\text{d}t}\] napięcie: \(u_L(t)=\omega LI_m\sin(\omega t + \theta_i + \frac{\pi}{2})\)
\(U_m = \omega LI_m\)
i \(\theta_u=\theta_i + \frac{\pi}{2}\)

Reaktancja indukcyjna/opór bierny

\(X_L=\omega L\)

Odwrotność reaktancji/susceptancja indukcyjna

\(B_L=\frac{1}{X_L} = \frac{1}{\omega L}\)

Postać zespolona

\[jU_{L_{rms}} = j \omega L I_{L_{rms}}\]

Impedancja - połączenie szeregowe

Abstrakcyjny element sieci, który zastępuje połączenie \(RLC\)
\(u(t)=|Z|I_m\sin(\omega t + \theta_i + \arctan(\frac{X}{R}))\)
\(|Z|=\sqrt{R^2 + X^2}\) gdzie \(X=X_L-X_C\) - wypadkowa reaktancja połączenia cewki i kondensatora

Impedancja - połączenie równoległe

Abstrakcyjny element sieci, który zastępuje połączenie \(RLC\)
\(u(t)=|Y|U_m\sin(\omega t + \theta_u + \arctan(\frac{B}{G}))\)
\(|Y|=\sqrt{G^2 + B^2}\) gdzie \(B=B_C-B_L\) - wypadkowa admitancja połączenia cewki i kondensatora

Wartość średnia

pole powierzchni obszaru, który wyznaczają opisana funkcja, granice jej okresu oraz oś czasu. \[F_{avg}=\frac{1}{T}\int_{t_1}^{t_1 + T}f(t)\text{d}t\]

Wartość skuteczna (rms)

\[F_{rms} = \sqrt{(f(t)^2)_{avg}} = \frac{F_m}{\sqrt{2}} = 0,707 I_m\]

Współczynnik kształtu

\[\frac{\text{wartosc skuteczna}}{\text{wartosc srednia}} = \frac{\frac{I_m}{\sqrt{2}}}{\frac{2}{\pi}I_m} = 1,11\]

Współczynnik szczytu

\[\frac{\text{wartosc szczytowa}}{\text{wartosc skuteczna}} = \frac{U_m}{U_{rms}} = \frac{U_m}{\frac{U_m}{\sqrt{2}}} = 1,414\]

Algebra zespolona

Liczby zepolone: \[z=x+jy\] j: \[j=\sqrt{-1}\] sprżeżenie: \[\overline{z}=\overline{x-jy} = x+yj=z\] moduł: \[|z|=\sqrt{x^2+y^2}\] postać trygonometryczna: \[\tan(\theta) = \frac{x}{y}\] \[z = |z|(\cos(\theta)+j\sin(\theta))\] postać wykładnicza: \[|z|\text{e}^{j \theta}\]
